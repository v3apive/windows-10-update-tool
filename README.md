# Windows 10 Update Tool v41.2

Simple utility to upgrade Windows 7 or 8 systems to Windows 10.
Утилита для удобного обновления систем Windows 7 или 8 до Windows 10.

---

## English Description
Add Windows 10 Update Tool with ISO handling features. This tool helps to migrate legacy systems to Windows 10 even if disk mounting is broken.

### Features:
- **ISO Download:** Direct high-speed link to optimized image.
- **Extraction:** Smart Robocopy engine for Windows 7 compatibility.
- **Disk Check:** Manual and automatic C: drive space verification.
- **Lite OS:** Clean distribution without bloatware.

**Note:** If `api-ms-win-core` errors occur, install Visual C++ from the `Required_Components` folder. Run as Administrator.

---

## Описание на русском
Программа для перехода с Windows 7/8 на Windows 10. Помогает обновиться даже на старых ПК, где не работает стандартное монтирование ISO-образов.

### Особенности:
- **Загрузка ISO:** Прямая ссылка на оптимизированный образ.
- **Распаковка:** Использование Robocopy для совместимости с Win 7.
- **Проверка места:** Контроль свободного пространства на диске C:.
- **Чистая ОС:** Образ без лишнего предустановленного софта.

**Важно:** Если при запуске возникла ошибка `api-ms-win-core`, установите компоненты из папки `Required_Components`. Запускать от имени Администратора.
